# RESTAURANTS REVIEW APP

A fully responsive app with offline capabilities to view reviews of restaurants near you. It uses Mapbox API to view map and advanced javascript features like service workers to cache the website and make it work even in offline or slow connections.

## FEATURES

* Fully responsive
* Fully accessible
* Offline capability

## HOW TO USE THIS APP

1) Download the zip of this repository

2) Extract the zip file and open the project location in your terminal by navigating to this project repository.

3) Start any local server at port 8000.

    > If you have php installed, then start the server by this command-
    >
    > ```php -S localhost:8000```

4) Open your chrome browser and type this URL ```http://localhost:8000/index.html``` in the search box.

5) The app should now be up and running in your browser. Enjoy...
